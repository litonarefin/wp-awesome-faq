== Description ==

Awesome Colorful FAQ WordPress Plugin Developed by [CodexCoder](http://www.codexcoder.com/ "CodexCoder").

Awesome Colorful FAQ WordPress Plugin that can help you to add FAQ (Frequently Asked Question) in your website.

It has custom Post type option to add FAQ. You can easily add these as like a normal post. You will find the shortcode facility to show add FAQ's into any page or post. Just add this code [colorful_faq] at the end of any page or post to show FAQ. 

This plugin made with Bootstrap 3 and flat ui colors.

Take Look our Video Tutorial:
[youtube http://youtu.be/A7L-xdXfuzk]

== Installation ==

1. Upload `ccr-colorful-faq-pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[colorful_faq]` in your page or post where you want to show FAQs

= How can i add YouTube or Vimeo video? =

Yes, you can add youtube or vimeo video link. To add video from youtube or vimeo copy tge video link from you tube and paste the link and faq content section.

You can see color selector right sidebar in faq adding section. There have some color name with preview you can select there.
